# Unit Converter

Basit bir Python birim dönüştürücü uygulaması.

Kilometre → Mil ve Celsius → Fahrenheit dönüşümleri yapar.

## Özellikler

* Kilometreyi mile çevirme
* Celsius'u Fahrenheit'a dönüştürme
* Kullanıcı dostu menü yapısı
* Geçersiz girişler için hata yakalama

## Kullanım

Programı çalıştırmak için:

```bash
python main.py
```

## Örnek Çalışma

```
--- Birim Dönüştürücü ---
1) Kilometre → Mil
2) Celsius → Fahrenheit
3) Çıkış
```
